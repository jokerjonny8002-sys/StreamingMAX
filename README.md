# StreamingMAX

**DJ and live-streaming mission control for macOS.**

StreamingMAX is a desktop application designed to help DJs and streamers check system performance, studio readiness, network quality, and essential applications before going live.

## Current Features

- Live CPU, RAM, disk, and performance gauges
- Stream Readiness Score
- Mission Scan preflight checklist
- OBS Studio detection
- Rekordbox detection
- Built-in network speed test
- DJ Mode, Stream Mode, and Clean Mode
- System recommendations
- Animated Cockpit v2 interface
- macOS DMG installer

## Built With

- Electron
- JavaScript
- HTML
- CSS
- Node.js
- Git

StreamingMAX is currently in public beta and under active development.
